Unit 6 : Creating Digital Graphics



For Unit 6, you are required to create 2 image products. One bitmap/raster graphic, using Adobe Photoshop. One vector graphic, using Adobe Illustrator.



Advertisement (Photoshop)


The raster image you are tasked with creating is a mobile phone advertisement.




User Guide (Illustrator)


The vector image tasked to you for creation is a User Guide for the same mobile phone. You can use a different phone if you want, but then you'll have to research multiple phones in order to get the relevant information.




Unit 10 : Database Development



Database (Access)


The Unit 10 assignment 1 was to design, develop and test a database for "Iechyd Surgery", a fictitious medical practice, using Microsoft's Access Databases application.

For developing the database, you are provided a spreadsheet containing data to use in the patient table. You would then import the data from the spreadsheet into the access database, in the form of a table.

Next, you would format the table fields to have appropriate datatypes and validation, as well as names and descriptions that fit what data the field represents.
